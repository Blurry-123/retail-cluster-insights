📊 Customer Segmentation using PCA and K-Means Clustering
🚀 Project Overview

This project focuses on discovering hidden patterns in unlabeled customer data using unsupervised machine learning techniques. The goal is to reduce high-dimensional data into meaningful components and group similar customers into distinct segments for better business decision-making.

We use Principal Component Analysis (PCA) for dimensionality reduction and K-Means Clustering for segmentation.

🎯 Objectives
Reduce dataset dimensions using PCA
Identify optimal number of clusters using:
Elbow Method
Silhouette Score
Apply K-Means Clustering
Interpret clusters as actionable customer personas
Extract business insights from grouped data
📂 Dataset
Dataset contains customer-related features (20+ columns)
Includes numerical and categorical attributes (preprocessed)
No target variable (unsupervised learning problem)
🛠️ Technologies Used
Python 🐍
Pandas
NumPy
Matplotlib / Seaborn
Scikit-learn
PCA (Principal Component Analysis)
K-Means Clustering
🧠 Methodology
1. Data Preprocessing
Handling missing values
Encoding categorical variables
Feature scaling (StandardScaler)
2. Dimensionality Reduction
Applied PCA
Reduced features from 20+ dimensions to 2D/3D space for visualization
3. Optimal Cluster Selection
Elbow Method to identify inertia drop
Silhouette Score to validate cluster quality
4. K-Means Clustering
Applied K-Means algorithm
Assigned cluster labels to each data point
5. Visualization
Scatter plots of PCA-reduced clusters
Cluster distribution analysis
📊 Results
Successfully grouped customers into meaningful clusters
Identified distinct behavioral patterns
Transformed raw data into business-friendly personas
💡 Key Insights
Customers show clear grouping patterns after dimensionality reduction
Optimal number of clusters improves interpretability
Each cluster represents a unique customer behavior segment
